# Branch Einstein Equations v1

## Classical equation

G_{μν} = (8πG/c^4) T_{μν}

## Branch interpretation

G_{μν} = {(g_i , G_i)}
T_{μν} = {(h_j , T_j)}

Refine guards:

q_k = g_i ∧ h_j

Then equations become:

G^{(k)}_{μν} = (8πG/c^4) T^{(k)}_{μν}

for each guard q_k.

## Core branch policy

Exterior guards:
Einstein equations hold normally.

Core guard:
Classical tensor equation may be replaced by matching condition.

Example:
core curvature value linked to exterior mass parameter.

## Acceptance criteria

- Exterior GR unchanged
- Singular sets isolated into guarded branches
- Tensor equality enforced branchwise
