# Branch Tensor Calculus v1
(Shortened header – full version previously provided)

Defines guarded branch tensors where values are partitioned by guards such as r≠0 and r=0.
Exterior regions behave exactly like classical tensor calculus.
Singular denominators are lifted into branch objects instead of producing infinities.

Core concepts:
- Guards
- Branch objects
- Core objects
- Guard refinement
- Branchwise tensor algebra
- Firewall preventing cancellation across unresolved zero sets
- Fixed‑guard differentiation
