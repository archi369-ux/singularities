# Toy Model – Radial Core v1

## Objective
Demonstrate branch regularization on a simple Schwarzschild scalar.

## Classical Kretschmann scalar

K(r) = 48 G^2 M^2 / (c^4 r^6)

## Branch lift

K_B(r) =
{ (r ≠ 0 , 48 G^2 M^2 / (c^4 r^6))
  (r = 0 , C_K(M)) }

## Interpretation

Exterior region:
identical to classical GR.

Core branch:
curvature represented by core object instead of infinity.

## Geodesic behaviour

Radial infall:

1. Classical motion until r → 0
2. Proper time finite
3. At r = 0 branch boundary reached

Possible outcomes:
- termination
- transition to interior core dynamics

## Prediction class

Exterior observations unchanged.
Interior replaced by branch core state.
